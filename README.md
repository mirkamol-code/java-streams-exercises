# java-streams-exercises

A hands-on Java project focused on mastering the **Stream API** through practical, real-world style exercises.  
The goal of this repository is to practice **functional and declarative programming** using Java Streams.

This project contains **20 practical exercises** based on a simple transactions dataset.  
It covers core Stream operations such as:

- `filter`

- `collect`
- `groupingBy`
- `distinct`

- Strengthening Java functional programming skills

## ðŸ§ª Exercise List

1. **Filter Transactions by Type**  
   Find all `"CREDIT"` transactions.

2. **Get Total Amount of All Transactions**  
   Calculate the total amount of all transactions.

3. **Find the Maximum Transaction Amount**  
   Find the transaction with the highest amount.

4. **Count Transactions by Type**  
   Count the number of `"DEBIT"` and `"CREDIT"` transactions.

5. **Get Distinct Customers**  
   Retrieve a list of distinct customers from the transactions.

6. **Find Transactions Above a Threshold**  
   Find all transactions with an amount greater than `300`.

7. **Group Transactions by Customer**  
   Group transactions by customer ID.

8. **Find the First Transaction**  
   Find the first transaction in the dataset.

9. **Check for Any High-Value Transaction**  
   Check if there is any transaction with an amount greater than `1000`.

10. **Sort Transactions by Amount**  
    Sort all transactions by amount in ascending order.

11. **Calculate Average Transaction Amount**  
    Find the average transaction amount.

12. **Find Transactions by Customer Name**  
    Find all transactions made by a specific customer (e.g., `"Alice"`).

13. **Remove Duplicate Transactions**  
    Create a list of transactions without duplicates.

14. **Join Customer Names**  
    Concatenate all customer names into a single string separated by commas.

15. **Find the Earliest Transaction**  
    Find the transaction with the earliest date.

16. **Filter Transactions by Year**  
    Find all transactions from the year `2024`.

17. **Calculate Total Amount by Customer**  
    Calculate the total transaction amount for each customer.

18. **Find Transactions Containing "DEBIT"**  
    Find all transactions where the type contains the substring `"DEBIT"`.

19. **Group Transactions by Type**  
    Group transactions into `"DEBIT"` and `"CREDIT"` categories.

20. **Find Transactions with Specific Email**  
    Find all transactions made by customers with an email containing `"example.com"`.
